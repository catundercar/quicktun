# quicktun Agent (Linux) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a Linux agent that runs on a bastion host, registers with the control plane, renders its own rathole-client.toml from a server-issued bootstrap response, and supervises rathole-client locally. Heartbeats keep `Site.last_seen_time` and `Site.hostname/os/agent_version` fresh.

**Out of scope (deferred):**
- Windows service (separate Plan 7.5 — same agent core, different process management & install).
- Operator CLI (Plan 9).
- Auth-proxy on port 443 (Plan 8 — until then, Phase-1 dev runs the agent against the gRPC port directly with rathole's control on a separate port the operator forwards).
- Deployment scripts / systemd unit (Plan 10).

**Token contract (decided in Plan 6.5):** Operator pastes the raw token from the install command into the agent config. Agent sends raw as `Authorization: Bearer <token>` to the control-plane gRPC API; computes `sha256_hex(raw)` and writes that into rathole-client.toml. Per `docs/07-token-contract.md`.

---

## Architecture

```
[Bastion host]                          [Control plane]
+--------------------+                  +--------------------+
| quicktun-agent     |                  | quicktun-server    |
|                    |                  |                    |
|  bootstrap loop    |  gRPC Bootstrap  |   AgentService     |
|  - read config ----┼─────────────────►│   - Bootstrap()    │
|  - render rcfg     │                  │   - Heartbeat()    │
|                    │                  │                    │
|  heartbeat loop    │  gRPC Heartbeat  │  agent token auth  │
|  - every 15s ------┼─────────────────►│  interceptor       │
|                    │                  │                    │
|  supervisor        │                  │  relay.Manager     │
|  - rathole-client ─┼──── tunnel ─────►│  - rathole-server  │
+--------------------+                  +--------------------+
```

The agent is a single Go binary. On start: parses YAML config (control endpoint, raw token, agent_id) → calls `AgentService.Bootstrap` (auth via raw token) → renders `<state_dir>/rathole-client.toml` → spawns `rathole --client <cfg>` via the existing `internal/supervisor/`. Then heartbeats every 15s.

If `Heartbeat` returns `should_rebootstrap=true`, agent re-fetches bootstrap and rewrites the rathole config; supervisor restart picks up the new tunnels.

```proto
syntax = "proto3";

package quicktun.v1;

import "google/api/annotations.proto";
import "google/protobuf/timestamp.proto";

option go_package = "github.com/tulip/quicktun/gen/go/quicktun/v1;quicktunv1";

service AgentService {
  // Bootstrap returns the agent's full operating state: rathole control
  // endpoint, per-service tunnel bindings, and a recommended heartbeat
  // interval. The agent calls this on start and again whenever Heartbeat
  // returns should_rebootstrap=true.
  rpc Bootstrap(BootstrapRequest) returns (BootstrapResponse) {
    option (google.api.http) = {
      post: "/v1/agent:bootstrap"
      body: "*"
    };
  }

  // Heartbeat reports liveness + host metadata. Server updates the
  // corresponding Site row's last_seen_time / hostname / os /
  // agent_version. Returns should_rebootstrap=true when bindings have
  // changed since the agent's last Bootstrap.
  rpc Heartbeat(HeartbeatRequest) returns (HeartbeatResponse) {
    option (google.api.http) = {
      post: "/v1/agent:heartbeat"
      body: "*"
    };
  }
}

message BootstrapRequest {
  string hostname = 1;
  string os = 2;
  string agent_version = 3;
}

message BootstrapResponse {
  // Site identity.
  string site_name = 1;       // e.g. projects/p1/sites/bastion-1
  string project_slug = 2;
  string site_slug = 3;

  // Rathole control endpoint.
  string rathole_control_addr = 4;  // e.g. relay.example.com:20000

  // Tunnels to expose. Service slugs match server-side rendered keys.
  repeated TunnelBinding tunnels = 5;

  // Recommended heartbeat interval (seconds).
  int32 heartbeat_seconds = 6;

  // Bootstrap version: heartbeat returns the same value when nothing
  // has changed; differing value means re-bootstrap.
  string config_version = 7;
}

message TunnelBinding {
  string service_slug = 1;        // e.g. ssh
  string target_addr = 2;         // local target, e.g. 127.0.0.1
  uint32 target_port = 3;         // local target port
  string proto = 4;               // tcp | udp (Phase 1: tcp only)
  uint32 relay_port = 5;          // for diagnostics (rathole picks the bind)
}

message HeartbeatRequest {
  string hostname = 1;
  string os = 2;
  string agent_version = 3;
  repeated string lan_cidrs = 4;  // optional
  string config_version = 5;      // last value the agent processed
}

message HeartbeatResponse {
  bool should_rebootstrap = 1;
  google.protobuf.Timestamp server_time = 2;
}
```

### Step 2: Agent token auth interceptor

Create `internal/auth/agent_interceptor.go`:

```go
package auth

import (
	"context"
	"crypto/sha256"
	"encoding/hex"
	"errors"
	"strings"
	"time"

	"google.golang.org/grpc"
	"google.golang.org/grpc/codes"
	"google.golang.org/grpc/metadata"
	"google.golang.org/grpc/status"
	"gorm.io/gorm"

	"github.com/tulip/quicktun/internal/model"
)

// AgentPrincipal identifies an authenticated site agent on the request ctx.
type AgentPrincipal struct {
	SiteID    uint64
	ProjectID uint64
}

type agentCtxKey struct{}

// AgentFromContext returns the principal if the request was authenticated as
// an agent.
func AgentFromContext(ctx context.Context) (*AgentPrincipal, bool) {
	a, ok := ctx.Value(agentCtxKey{}).(*AgentPrincipal)
	return a, ok
}

// AgentInterceptor authenticates Agent* RPCs via Bearer site agent tokens.
// Methods not in the agent allowlist pass through (operator interceptor
// handles those).
func AgentInterceptor(db *gorm.DB) grpc.UnaryServerInterceptor {
	return func(ctx context.Context, req any, info *grpc.UnaryServerInfo, handler grpc.UnaryHandler) (any, error) {
		// Only intercept AgentService methods.
		if !strings.HasPrefix(info.FullMethod, "/quicktun.v1.AgentService/") {
			return handler(ctx, req)
		}

		md, ok := metadata.FromIncomingContext(ctx)
		if !ok {
			return nil, status.Error(codes.Unauthenticated, "missing metadata")
		}
		raw, err := bearerFromMD(md)
		if err != nil {
			return nil, status.Error(codes.Unauthenticated, err.Error())
		}

		hash := sha256.Sum256([]byte(raw))
		hex := hex.EncodeToString(hash[:])

		var tok model.SiteAgentToken
		err = db.WithContext(ctx).
			Where("token_hash = ?", hex).
			First(&tok).Error
		if errors.Is(err, gorm.ErrRecordNotFound) {
			return nil, status.Error(codes.Unauthenticated, "invalid token")
		}
		if err != nil {
			return nil, status.Errorf(codes.Internal, "auth lookup: %v", err)
		}
		if tok.ExpireTime != nil && tok.ExpireTime.Before(time.Now()) {
			return nil, status.Error(codes.Unauthenticated, "token expired")
		}

		var site model.Site
		if err := db.WithContext(ctx).First(&site, tok.SiteID).Error; err != nil {
			return nil, status.Errorf(codes.Internal, "site lookup: %v", err)
		}

		ctx = context.WithValue(ctx, agentCtxKey{}, &AgentPrincipal{
			SiteID:    site.ID,
			ProjectID: site.ProjectID,
		})
		return handler(ctx, req)
	}
}

func bearerFromMD(md metadata.MD) (string, error) {
	auths := md.Get("authorization")
	if len(auths) == 0 {
		return "", errors.New("missing authorization")
	}
	parts := strings.SplitN(auths[0], " ", 2)
	if len(parts) != 2 || !strings.EqualFold(parts[0], "Bearer") {
		return "", errors.New("malformed authorization")
	}
	return parts[1], nil
}
```

	}
	return d.db.WithContext(ctx).Model(&model.Site{}).
		Where("id = ?", siteID).
		Updates(updates).Error
}

```go
package grpcsvc

import (
	"context"
	"crypto/rand"
	"encoding/hex"
	"strconv"
	"time"

	"google.golang.org/grpc/codes"
	"google.golang.org/grpc/status"
	"google.golang.org/protobuf/types/known/timestamppb"

	quicktunv1 "github.com/tulip/quicktun/gen/go/quicktun/v1"
	"github.com/tulip/quicktun/internal/auth"
	"github.com/tulip/quicktun/internal/dao"
	"github.com/tulip/quicktun/internal/model"
	"github.com/tulip/quicktun/internal/resource"
)

const heartbeatSeconds = 15

type AgentService struct {
	quicktunv1.UnimplementedAgentServiceServer
	projects *dao.ProjectDAO
	sites    *dao.SiteDAO
	services *dao.ServiceDAO
	tokens   *dao.SiteAgentTokenDAO
	relayHost string // e.g. "relay.example.com" — Project's RelayAddr without port
}

func NewAgentService(p *dao.ProjectDAO, s *dao.SiteDAO, svc *dao.ServiceDAO, tk *dao.SiteAgentTokenDAO, relayHost string) *AgentService {
	return &AgentService{projects: p, sites: s, services: svc, tokens: tk, relayHost: relayHost}
}

func (a *AgentService) Bootstrap(ctx context.Context, req *quicktunv1.BootstrapRequest) (*quicktunv1.BootstrapResponse, error) {
	pr, ok := auth.AgentFromContext(ctx)
	if !ok {
		return nil, status.Error(codes.Unauthenticated, "no agent principal")
	}

	site, err := a.sites.FindByID(ctx, pr.SiteID)
	if err != nil { return nil, status.Errorf(codes.Internal, "site: %v", err) }

	project, err := a.projects.FindByID(ctx, pr.ProjectID)
	if err != nil { return nil, status.Errorf(codes.Internal, "project: %v", err) }

	minP, _, err := resource.ParsePortRange(project.RelayPortRange)
	if err != nil { return nil, status.Errorf(codes.FailedPrecondition, "port range: %v", err) }

	if err := a.sites.UpdateAgentMeta(ctx, site.ID, req.Hostname, req.Os, req.AgentVersion, nil); err != nil {
		return nil, status.Errorf(codes.Internal, "update meta: %v", err)
	}

	svcs, err := a.services.ListBySite(ctx, site.ID, 1000, "")
	if err != nil { return nil, status.Errorf(codes.Internal, "services: %v", err) }

	tunnels := make([]*quicktunv1.TunnelBinding, 0, len(svcs))
	for _, s := range svcs {
		if s.RelayPort == nil { continue }
		tunnels = append(tunnels, &quicktunv1.TunnelBinding{
			ServiceSlug: s.Name,
			TargetAddr:  s.TargetAddr,
			TargetPort:  uint32(s.TargetPort),
			Proto:       string(s.Proto),
			RelayPort:   uint32(*s.RelayPort),
		})
	}

	cv := computeConfigVersion(svcs)

	return &quicktunv1.BootstrapResponse{
		SiteName:           resource.FormatSiteName(project.Slug, site.Name),
		ProjectSlug:        project.Slug,
		SiteSlug:           site.Name,
		RatholeControlAddr: a.relayHost + ":" + strconv.Itoa(int(minP)),
		Tunnels:            tunnels,
		HeartbeatSeconds:   heartbeatSeconds,
		ConfigVersion:      cv,
	}, nil
}

func (a *AgentService) Heartbeat(ctx context.Context, req *quicktunv1.HeartbeatRequest) (*quicktunv1.HeartbeatResponse, error) {
	pr, ok := auth.AgentFromContext(ctx)
	if !ok {
		return nil, status.Error(codes.Unauthenticated, "no agent principal")
	}

	if err := a.sites.UpdateAgentMeta(ctx, pr.SiteID, req.Hostname, req.Os, req.AgentVersion, req.LanCidrs); err != nil {
		return nil, status.Errorf(codes.Internal, "update meta: %v", err)
	}

	svcs, err := a.services.ListBySite(ctx, pr.SiteID, 1000, "")
	if err != nil { return nil, status.Errorf(codes.Internal, "services: %v", err) }
	current := computeConfigVersion(svcs)

	return &quicktunv1.HeartbeatResponse{
		ShouldRebootstrap: req.ConfigVersion != current,
		ServerTime:        timestamppb.Now(),
	}, nil
}

// computeConfigVersion is a stable hash over the (service_slug, relay_port,
// target_addr, target_port, proto) tuples for a site. Any change → version
// changes → next heartbeat tells agent to rebootstrap.
func computeConfigVersion(svcs []model.Service) string {
	// sort by slug for stability, then sha256 the canonical concat.
	// (Implementer should write this — keep it deterministic and short.)
	// Return hex of first 8 bytes of sha256.
	// ...
	_ = svcs
	b := make([]byte, 8)
	rand.Read(b)
	return hex.EncodeToString(b) // placeholder: implementer replaces with sha256

### Step 2: rathole-client.toml renderer

```go
// internal/agent/render.go
package agent

import (
	"crypto/sha256"
	"encoding/hex"
	"fmt"
	"strings"

	quicktunv1 "github.com/tulip/quicktun/gen/go/quicktun/v1"
)

// RenderRatholeClient writes a rathole-client TOML config from a Bootstrap
// response. The token field is the sha256 hex of rawToken (per token contract).
func RenderRatholeClient(boot *quicktunv1.BootstrapResponse, rawToken string) (string, error) {
	if boot.RatholeControlAddr == "" {
		return "", fmt.Errorf("agent: missing rathole_control_addr")
	}
	hash := sha256.Sum256([]byte(rawToken))
	tokenHex := hex.EncodeToString(hash[:])

	var b strings.Builder
	fmt.Fprintf(&b, "# quicktun-agent rendered config for %s\n", boot.SiteName)
	fmt.Fprintf(&b, "# DO NOT EDIT MANUALLY.\n\n")
	b.WriteString("[client]\n")
	fmt.Fprintf(&b, "remote_addr = %q\n\n", boot.RatholeControlAddr)

	for _, t := range boot.Tunnels {
		// rathole's client-side service key matches the server's flattened key
		// (see internal/relay/render.go): "<site_slug>__<service_slug>".
		name := boot.SiteSlug + "__" + t.ServiceSlug
		fmt.Fprintf(&b, "[client.services.%s]\n", name)
		fmt.Fprintf(&b, "token = %q\n", tokenHex)
		fmt.Fprintf(&b, "local_addr = \"%s:%d\"\n\n", t.TargetAddr, t.TargetPort)
	}
	return b.String(), nil
}
```

Test: golden — verify two services + token hex + control endpoint all appear.

func (r *Runtime) Run(ctx context.Context) error {
	// 1. Bootstrap with retry
	// 2. Render rathole-client.toml to <state_dir>/rathole-client.toml
	// 3. Spawn supervisor for cfg.RatholeBinary cfg.RatholeArgs <path>

	// 5. On ctx.Done: stop supervisor and return
}

Server binds 127.0.0.1; agent connects to `<RelayAddr>:<port>` where RelayAddr is the publicly-reachable hostname. The auth-proxy in Plan 8 will be the layer that maps public 443 → 127.0.0.1:<port> on the server. Pre-Plan-8: operator opens the rathole port directly OR runs both ends on the same network (dev).
- `auth.AgentFromContext` is the agent equivalent of the existing operator `auth.OperatorFromContext` (or whatever it's named — read existing).

**Open questions left for Plan 8/9:**
- How does the agent learn the auth-proxy public address for the rathole tunnel? Currently `RatholeControlAddr` is rendered from the server's `RelayAddr` config string. Plan 8 may want to swap that for an auth-proxy-fronted endpoint with a per-tunnel sub-domain or path.
- Should agent token rotation trigger a forced rebootstrap? Today, the token hash in rathole config matches what's in DB; if operator rotates via `RotateSiteAgentToken`, the rendered server-side rathole config changes (Plan 6 hooks Refresh) but the agent only learns about it at next heartbeat (which now fails because its token is invalid). Options: (a) operator coordinates rotation with agent restart, (b) Heartbeat returns "token will rotate at T" advance notice. Defer to Plan 9.
